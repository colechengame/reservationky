# 京都堂預約派號系統流程報告

叫叫我雲端叫號機 × 中醫預約管理流程報告

## 功能特色

- 📊 系統概覽：導入目的與效益分析
- 🔄 完整流程：從預約到派號的完整時間線
- 📌 派號規則：詳細的派號邏輯與離池判定
- 🚀 延展佈署：其他門市佈署前置作業說明
- 🎮 互動模擬：可實際操作的派號模擬器
- 🧪 測試案例：10個完整測試案例驗證

## 技術特點

- 純 HTML + CSS + JavaScript
- 無需安裝任何依賴
- 可直接在瀏覽器中運行
- 中醫配色主題設計
- 響應式設計，支援手機、平板、電腦

## 使用方式

直接開啟 `預約派號流程報告.html` 即可使用。
